# minishell

너무 정신없어서 폴더 정리 했음.  
Makefile / includes / srcs  
  ㄴ srcs 아래  
      ㄴ built_in_functions : 빌트인 함수들  
      ㄴ error : 에러 함수 (아직 별거 없음)  
      ㄴ parse : envp 파싱하는 함수 넣어놓음.  
      ㄴ temp : 터미널 관련 함수나 파일 접근 권한 함수 등 아직 안한 부분 함수들 temp에 넣어놓음.  
      ㄴ utils : 파싱할 때 필요한 자잘한 함수들 넣어놓음.
